# Wine ML Comparison Project 🍷

A professional machine learning project focused on classifying wine cultivars based on their unique chemical signatures.

## Overview
This project uses the **UC Irvine Wine dataset** to develop a classification system. We follow a clear, aesthetically pleasing **8-step pipeline** inspired by high-quality Kaggle notebooks.

## The 8-Step Pipeline
1. **🍷 Project Initiation** — Loading and labeling the constituents.
2. **📊 Data Summary** — Identifying feature variance.
3. **📈 Distribution Analysis** — Understanding feature skewness.
4. **🔥 Correlation & Heatmap** — Finding chemical relationships (e.g., Flavanoids vs. Phenols).
5. **📦 Outlier & Class Separation** — Identifying Proline as a key differentiator.
6. **🗺️ Multivariate Relationships** — Visualizing class clusters.
7. **🛠️ Machine Learning Setup** — Implementing an 80/20 split and scaling pipeline.
8. **🚀 Algorithm Evaluation** — Comparing LR, KNN, and Random Forest.

## Current Progress
We have completed the **EDA and Data Preparation** phases and are currently in the **Model Evaluation** stage.

## How to Run
1. Ensure Python and Jupyter are installed.
2. Install dependencies:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
3. Open `Wine_ML_Comparison.ipynb` to see the full, insight-rich analysis.
